# Campus & Playground Photos

Drop your campus/playground/building photo files directly into this folder
(`.jpg`, `.jpeg`, `.png`, or `.webp`). They will automatically appear in the
"Campus & Playground" section on the Gallery page — no code changes needed.

Tips:

- Use descriptive file names (e.g. `playground-1.jpg`, `main-building.jpg`) —
  the name (minus the extension and any numbers/dashes) is used as the photo's
  caption.
- Images don't need to be perfectly edited/cropped — the gallery grid crops
  them to a consistent size automatically.
- Once Cloudinary is configured, you can instead upload photos via the admin
  panel's Gallery section for full CRUD management. This folder is a quick,
  no-setup alternative for photos you already have on hand.
